# Netflix Data Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, Matplotlib, and Power BI.

## Objectives
- Clean the dataset
- Analyze Netflix content
- Visualize trends
- Build an interactive dashboard

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Power BI

## Dataset
Netflix Movies and TV Shows Dataset

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Movies vs TV Shows
- Top Countries
- Top Genres
- Ratings Distribution
- Movies Released by Year

## Key Insights
- Netflix contains more Movies than TV Shows.
- The United States contributes the highest number of titles.
- International Movies are among the most common genres.
- TV-MA is the most frequent content rating.
